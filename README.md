### How to install
Import the *database.sql* to your SQL database
Drag and drop the server upload files to your server, and edit databases.cfg to have this.
```
	"timer"
	{
		"driver"		"default"
		"host"			"hostnamehere"
		"database"		"dbnamehere"
		"user"			"usernamehere"
		"pass"			"passwordhere"
		//"timeout"		"0"
		"port"		   	"porthere"
	}
```
